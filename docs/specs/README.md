## 系统依赖关系

```
指令系统
  ├── 场景上下文系统（构建 SceneContext）
  ├── 横切支撑系统（指令门控 / Vital / ABL / 刻印 / TALENT 修正）
  ├── 结算系统（SOURCE → 属性变化）
  │   └── 关系系统（结算后更新阶段）
  ├── 事件系统（匹配触发事件）
  │   └── 关系系统（阶段 rank 比较）
  ├── 口上系统（选择对话文本）
  │   └── 事件系统（事件 key 优先于指令 key）
  ├── 标记系统（施加标记）
  ├── 同行与约会系统（执行操作）
  └── 地图与导航系统（地点标签匹配）

地图与导航系统
  └── 同行与约会系统（同行角色跟随移动）

日程系统
  └── 同行与约会系统（同行/约会角色跳过刷新）

体力与气力系统
  ├── 指令系统（动作消耗）
  ├── 结算系统（恢复/扣除落地）
  ├── 日程系统（疲劳返家与休憩）
  ├── 同行与约会系统（同行/约会中消耗修正）
  └── 场景上下文系统（状态门槛判定）

横切支撑系统
  ├── 指令系统（命令可用性门控）
  ├── 结算系统（DOWNBASE / SOURCE_EXTRA / ABL 升级 / 刻印）
  ├── 关系系统（亲密值参与阶段推算）
  └── 体力与气力系统（恢复与门槛）
```
