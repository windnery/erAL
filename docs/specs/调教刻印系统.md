# 调教刻印系统

## 概述

刻印是调教过程中自动产出的长期标记，反映角色在调教中的累积经历。它直接复用现有 mark 基础设施，但在结算后由 `TrainingService` 自动施加，而非由命令手动指定。

erAL 不含时姦刻印（无时停系统）。刻印类型仅限快乐、屈服、苦痛三种。

## 设计目标

- 让调教行为产生可感知的长期影响
- 刻印等级影响后续调教的 SOURCE 倍率和口上分支
- 复用 `marks.toml` + `CharacterState.marks`，不引入新的状态存储

## 刻印类型

| 刻印 key | 显示名 | max_level | 触发条件 | 长期效果 |
|----------|--------|-----------|---------|---------|
| `mark_pleasure` | 快乐刻印 | 3 | 高潮时自动施加（单次高潮 +1） | 提升对应部位快感 SOURCE 倍率 |
| `mark_submission` | 屈服刻印 | 3 | 屈服/奉仕类行为后 submission PALAM 达到阈值 | 降低拒绝概率，解锁主动请求 |
| `mark_pain` | 苦痛刻印 | 3 | pain PALAM 达到阈值时施加 | 影响恐惧/服从 SOURCE |

### 刻印等级与效果

#### 快乐刻印

| 等级 | 触发 | 效果 |
|------|------|------|
| 1 | 累计 1 次高潮 | 所有 pleasure_* SOURCE ×1.2 |
| 2 | 累计 5 次高潮 | 所有 pleasure_* SOURCE ×1.5 |
| 3 | 累计 15 次高潮 | 所有 pleasure_* SOURCE ×2.0 |

#### 屈服刻印

| 等级 | 触发 | 效果 |
|------|------|------|
| 1 | submission PALAM 累计达到 3000 | 拒绝判定阈值放宽（spirit ≤ 5 才可能拒绝） |
| 2 | submission PALAM 累计达到 10000 | 解锁口上分支"主动请求"；奉仕 SOURCE ×1.3 |
| 3 | submission PALAM 累计达到 30000 | 角色可能在调教中主动发起行为（需反击系统） |

#### 苦痛刻印

| 等级 | 触发 | 效果 |
|------|------|------|
| 1 | 单次结算 pain PALAM ≥ 2000 | fear SOURCE ×1.2 |
| 2 | 单次结算 pain PALAM ≥ 5000 | fear SOURCE ×1.5，obedience SOURCE ×1.2 |
| 3 | 单次结算 pain PALAM ≥ 10000 | fear SOURCE ×2.0，obedience SOURCE ×1.5 |

## 数据流

```
调教命令执行
  → SOURCE 写入
  → SettlementService.settle_actor()
  → TrainingService.detect_results()
     → 检测高潮/拒绝/中断
     → 检测刻印触发条件
     → actor.add_mark() 施加或升级刻印
  → 刻印等级写入 actor.marks
  → 下次调教时，talent_effects 或 source_extra 根据 mark 等级调整 SOURCE
```

## 与现有系统的关系

- **标记系统**：刻印就是 mark，走 `marks.toml` 定义 + `CharacterState.marks` 存储
- **素质效果系统**：刻印等级可通过 `talent_effects.toml` 影响 SOURCE 倍率（新增刻印效果条目）
- **调教结算**：`detect_results()` 扩展为同时检测刻印
- **口上系统**：刻印可作为 `required_marks` 条件参与口上分支
- **存档系统**：刻印作为 marks 的一部分已自动可存档

## 实施范围

### 第一步：marks.toml + detect_results 扩展

1. 在 `marks.toml` 中新增三个刻印定义（group = "training"）
2. 扩展 `TrainingService.detect_results()` 增加刻印检测逻辑
3. 在 `talent_effects.toml` 中新增刻印对 SOURCE 的影响规则
4. 补充测试

### 后续扩展

- 刻印驱动的口上分支（角色专属）
- 屈服刻印 3 级与反击系统联动
- 刻印视觉展示（PALAM 面板中显示刻印等级）
