# 关系扩展系统

## 概述

关系扩展系统是在当前 5 段关系阶段之上的更深层规则层，用来承接誓约之后、道具门槛、历史门槛和更强的关系差分。

## 当前状态

- 当前 [关系系统](关系系统.md) 已经能覆盖基础阶段推算。
- 还没有把阶段后的长期分支、道具门槛和历史门槛系统化。

## 建议数据结构

### RelationshipBranch

| 字段 | 类型 | 说明 |
|------|------|------|
| `key` | str | 分支 key |
| `stage_key` | str | 绑定的关系阶段 |
| `required_items` | [str] | 需要的道具 |
| `required_memory` | [str] | 需要的记忆条件 |
| `effects` | dict[str, int] | 分支效果 |

## 建议流程

1. 在关系阶段满足后，继续判断分支条件。
2. 允许基于道具、事件历史、标记和角色身份做更细分的差分。
3. 分支结果影响指令、事件和口上。
