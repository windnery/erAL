# 商店与经济系统

## 概述

商店与经济系统负责货币、购买、出售、库存刷新和价格变化。它通常和物品系统绑定，但在玩法上是独立层：一个管“有什么”，一个管“怎么买到”。

## 当前状态

## 建议数据结构

### CurrencyState

| 字段 | 类型 | 说明 |
|------|------|------|
| `juel` | int | 当前货币 |
| `daily_income` | int | 日常收入 |
| `daily_expense` | int | 日常支出 |

### ShopDefinition

| 字段 | 类型 | 说明 |
|------|------|------|
| `key` | str | 商店 key |
| `display_name` | str | 显示名称 |
| `location_key` | str | 所在地点 |
| `open_slots` | [str] | 营业时段 |
| `stock_rules` | dict[str, int] | 商品与库存 |

## 建议流程

1. 进入商店地点后读取营业状态。
2. 根据时段和库存决定可购商品。
3. 购买后扣除货币、增加库存或物品。
4. 出售时把物品转成货币。
5. 价格变化可以由天气、关系或活动修正。
