# eraTW 状态编号全表

## 1. 目的

这份文档是 `eraTW` 状态变量的全量编号附录，优先覆盖：

- `CFLAG.csv`
- `TFLAG.csv`
- `FLAG.csv`

用途：

- 作为后续 `erAL` 迁移时的编号词典
- 避免实现时反复猜某个编号的含义
- 为后续整理 `BEFORETRAIN / EVENTCOMEND / DATE_CMN` 规则矩阵提供变量基础

- 注释列优先保留原 CSV 紧邻注释。
- 混入的后期补丁字段一并保留，因为它们同样占用了真实编号。
