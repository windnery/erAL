## 4. 使用建议

- `Train.csv` 的编号区间本身就是系统分层依据，迁移时应尽量保留“按区间分组”的组织方法。
- 对 `erAL` 来说，优先搬：
  - `300-399` 日常互动
  - `400-499` 生活/工作/移动
  - `580-699` 约会
- 不必在前期把 `0-199` 的深层 H 指令全部照搬。
